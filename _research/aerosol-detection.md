## Dust Aerosol Detection with Ensemble Machine Learning

**Domain:** Remote Sensing & Environmental Science  
**Institution:** Baylor University  
**Duration:** 2021-2023

### Overview

This research investigated dust aerosols using remote sensing techniques and developed ensemble deep learning models to analyze aerosols at multiple resolutions. The project utilized NASA MODIS satellite observations to create advanced tools for environmental monitoring and aerosol detection.

### Research Objectives

- Develop automated aerosol detection systems using satellite imagery
- Create multi-resolution analysis tools for comprehensive aerosol monitoring
- Apply ensemble deep learning for improved detection accuracy
- Contribute to environmental monitoring and climate research

### Methodology

**Data Acquisition:**
- Processed satellite data from NASA MODIS (Moderate Resolution Imaging Spectroradiometer)
- Collected atmospheric and environmental contextual data
- Prepared multi-spectral image datasets at various spatial resolutions
- Implemented data augmentation for robust model training

**Model Architecture:**
- Designed ensemble deep learning models combining:
  - Convolutional Neural Networks (CNNs) for spatial feature extraction
  - Residual Networks (ResNet) for deep feature learning
  - U-Net architectures for segmentation tasks
- Implemented multi-scale processing pipelines
- Developed custom loss functions for aerosol detection

**Analysis Tools:**
- Created low-resolution analysis tools for broad-area monitoring
- Developed high-resolution tools for detailed aerosol characterization
- Implemented visualization dashboards for data exploration
- Built preprocessing pipelines for satellite imagery

### Key Contributions

- Developed ensemble models achieving high accuracy in aerosol detection
- Created tools for analyzing aerosols at multiple spatial resolutions
- Applied computer vision techniques to environmental monitoring challenges
- Demonstrated practical applications of deep learning in remote sensing

### Research Publication

This work resulted in a peer-reviewed publication:

**📄 [Enhancing the Resolution of Satellite Imagery Using a Generative Model](https://ieeexplore.ieee.org/abstract/document/9799082)**  
*IEEE Symposium Series on Computational Intelligence (SSCI), 2022*

The paper presents a generative model approach for enhancing satellite imagery resolution, enabling more detailed aerosol analysis and environmental monitoring.
