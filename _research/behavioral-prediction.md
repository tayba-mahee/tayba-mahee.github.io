## Behavioral Feature Prediction Using Machine Learning

**Domain:** Computational Biology & Machine Learning  
**Institution:** Baylor University  
**Duration:** 2021-2023

### Overview

This research project focused on developing supervised machine learning models to predict behavioral features using bone marker data from primate subjects. The work bridged computational biology and machine learning to provide insights into behavioral analysis through quantitative skeletal measurements.

### Research Objectives

- Apply machine learning techniques to biological datasets for behavioral prediction
- Identify significant skeletal markers correlated with specific behaviors
- Develop robust predictive models with high accuracy and generalizability
- Collaborate with biology researchers to interpret findings in biological context

### Methodology

**Data Collection & Preprocessing:**
- Collected bone marker measurements from primate skeletal specimens
- Performed data cleaning and normalization
- Handled missing data through imputation techniques
- Created training and testing datasets with proper stratification

**Feature Engineering:**
- Analyzed correlation between bone markers and behavioral features
- Selected relevant features using statistical methods
- Applied dimensionality reduction techniques (PCA)
- Created derived features based on domain knowledge

**Model Development:**
- Implemented multiple supervised learning algorithms:
  - Random Forest Classifiers
  - Support Vector Machines (SVM)
  - Gradient Boosting Models
  - Neural Networks
- Performed hyperparameter tuning using grid search and cross-validation
- Evaluated models using accuracy, precision, recall, and F1-score

### Key Findings

- Identified specific bone markers with high predictive power for behavioral features
- Achieved significant prediction accuracy across multiple behavioral categories
- Demonstrated the feasibility of using skeletal data for behavioral analysis
- Provided biological insights validated by domain experts

### Technical Skills Developed

- Supervised machine learning (scikit-learn, Python)
- Feature selection and engineering
- Cross-validation and model evaluation
- Statistical analysis and hypothesis testing
- Collaboration with interdisciplinary research teams

### Impact

This research contributed to the understanding of the relationship between skeletal morphology and behavior in primates, with potential applications in:
- Evolutionary biology research
- Forensic anthropology
- Wildlife behavioral studies
- Conservation biology
