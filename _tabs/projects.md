## Featured Projects

### Cross-Cultural Information Seeking Behavior Study
**Master's Thesis Research**

Conducted comprehensive research examining how cultural differences impact information-seeking behavior and information recollection on websites. The study involved designing and executing user studies with participants from diverse cultural backgrounds to analyze website navigation patterns and information retention.

**Key Contributions:**
- Designed culturally-sensitive user study protocols
- Applied qualitative and quantitative analysis methods
- Utilized statistical tools to identify significant cultural factors
- Developed insights for culturally adaptive website designs using Fitts' Law

### Baylor Navigation System
**Full-Stack Web Application**

A comprehensive navigation and resource management system for Baylor University providing easy access to buildings and rooms across campus.

**Technologies:** React.js, Spring Boot, PostgreSQL, LeafletJS

**Features:**
- Interactive map-based interface using LeafletJS
- Internal navigation system for building and room locations
- Robust backend with Spring Boot and PostgreSQL
- Responsive frontend with React.js
- Collaborative development using Agile methodology

### AR Fishing
**Augmented Reality Application**

An innovative augmented reality fishing tool that fuses the traditional fishing experience with modern AR technology to promote sustainable fishing practices.

**Key Features:**
- Real-time fish classification with annotated information
- Location-based data integration
- Machine learning integration for species identification
- Focus on reducing overfishing and environmental conservation

**Research Focus:** Embedding machine learning with user experience in AR applications
